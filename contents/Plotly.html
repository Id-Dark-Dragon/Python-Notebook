Here is the function to make our chart:

# x and y are positional arguments not keyword.
plt.plot( x , y )                # linear graph     
plt.scatter( x , y )             # dotted graph 
plt.bar( x , y )                 # bar graph

To make multiple graphs overlap , we need to make a for loop: (in this eamples we are plotting the popularity of different programming languages from a pandas DF)


for column in df.columns:    # here, columns are programming languages names and indexes are dates.          
    plt.plot(df.index, df[column], 
             linewidth=3, label=df[column].name)
plt.legend(fontsize=16)      # this line is for making a guide for our graphs.

Adding Locator Tick Marks: (they are the little dashes between the main numbers in main axis that can ease the recognition.)


# In this example we have time on x-axis, and we want to add ticks between the years.
# Each tick presents a month.

import matplotlib.dates as mdates

# Create locators for ticks on the time axis
years = mdates.YearLocator()
months = mdates.MonthLocator()
years_fmt = mdates.DateFormatter("%Y")

# Here comes the code of the drawing chart...
# and then

ax1.xaxis.set_major_locator(years)
ax1.xaxis.set_major_formatter(years_fmt)
ax1.xaxis.set_minor_locator(months)

plt.show()
