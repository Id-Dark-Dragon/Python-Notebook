Matplotlib

Matplotlib is a comprehensive Python library for creating static, animated, and interactive visualizations.  Docs (and cheatsheets)

import matplotlib.pyplot as plt

CHARTS:

we can make data charts and graphs by using the combination of Pandas and Matplotlib. First, see some unnecessary configurations: 


plt.figure(figsize=(20,10))            # Size of graph.
plt.xticks(fontsize=14, rotation=45)   # x (and y below) ticks settings.
plt.yticks(fontsize=14)
plt.ylabel("popularity",fontsize=16)   # x (and y below) label settings.
plt.xlabel("time", fontsize=16)
plt.ylim(0,35000)                      # Range of numbers in axis

Here is the function to make our chart:

# x and y are positional arguments not keyword.
plt.plot( x , y )                # linear graph     
plt.scatter( x , y )             # dotted graph 
plt.bar( x , y )                 # bar graph

To make multiple graphs overlap , we need to make a for loop: (in this eamples we are plotting the popularity of different programming languages from a pandas DF)


for column in df.columns:    # here, columns are programming languages names and indexes are dates.          
    plt.plot(df.index, df[column], 
             linewidth=3, label=df[column].name)
plt.legend(fontsize=16)      # this line is for making a guide for our graphs.

To make our graphs smoother we can use the Rolling Average of values instead of the originals. read about it on Pandas notes.
