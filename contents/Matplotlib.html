Matplotlib

Matplotlib is a comprehensive Python library for creating static, animated, and interactive visualizations.  Docs (and cheatsheets)

We can make data charts and graphs by using the combination of Pandas and Matplotlib.

import matplotlib.pyplot as plt

NOTICE:  In Matplotlib we work with axes and figures. Think figure as the "window" that plot and its grids (axe) are in. Each axe can contain multiple plots. 

"The figure is the window and the axe is containing plots" . 

(Here see a fig with 2 axes) - (Here see 2 figs each with 1 axe)

There are two ways to make a plot in matplotlib, first and easier way is by using "plt.***()" which makes a (1 fig 1 axe) plot by default, 

and second and better is to use "subplot" which makes you able to customize your number of figs and axes more.

FIRST METHOD:


# some unnecessary configurations
plt.figure(figsize=(20,10), dpi=200)   # Size of graph.
plt.xticks(fontsize=14, rotation=45)   # x (and y below) ticks settings.
plt.yticks(fontsize=14)
plt.ylabel("popularity",fontsize=16)   # x (and y below) label settings.
plt.xlabel("time", fontsize=16)
plt.ylim(0,35000)                      # Range of numbers in axis.


# Here is the function to make our chart.
# x and y are positional arguments not keyword.
plt.plot( x , y )                # linear graph
plt.scatter( x , y )             # dotted graph
# If you write those three lines of code above together, It makes multiple graphs overlapping each other.
 

SECOND METHOD:

 (it does the same with the above code snippet)

fig, ax1 = plt.subplots()

ax1.plot(x, y)
ax1.scatter(x, y, label="plot")

fig.set_size_inches(20, 10)                                 # Size of graph.
ax1.tick_params(axis='x', labelsize=12, labelrotation=45)   # instead of "xticks()"
ax1.tick_params(axis='y', labelsize=12)
ax1.set_ylabel("popularity", fontsize=16)
ax1.set_xlabel("time", fontsize=16)
ax1.set_title("my plot")
ax1.set_ylim(0,35000)

plt.show()

Playing with figures and axes :


# One fig with two axes in two rows.
fig, (ax1, ax2) = plt.subplots(nrows=2, ncols=1, sharex=True)
                        # "sharex" determines if you want to repeat x-ticks in both axes or not. (also "sharey")


# One fig with two axes overlapping each other.
fig, ax1 = plt.subplots()
ax2 = ax1.twinx()       # double it


# two different figs each one axe.
fig1, ax1 = plt.subplots()
fig2, ax2 = plt.subplots()

Adding Locator Tick Marks: (they are the little dashes between the main numbers in main axis that can ease the recognition.)


# In this example we have time on x-axis, and we want to add ticks between the years.
# Each tick presents a month.

import matplotlib.dates as mdates

# Create locators for ticks on the time axis
years = mdates.YearLocator()
months = mdates.MonthLocator()
years_fmt = mdates.DateFormatter("%Y")    # "%Y" gives you just the year out of the date.

# Here comes the code of the drawing chart...
# and then

ax1.xaxis.set_major_locator(years)
 ax1.xaxis.set_major_formatter(years_fmt)  # showing years as the major ticks labels.
ax1.xaxis.set_minor_locator(months)
