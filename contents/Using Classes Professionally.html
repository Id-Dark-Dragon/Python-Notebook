Using Classes Professionally

If you integrated a library with your own-made class, you just need to make a object out of your class, not that library.

Here is a code without OOP:


from turtle import Turtle

# Making object
tim = Turtle()

.Here is the code using OOP ability:

# player.py
from turtle import Turtle


class Player(Turtle):       # Passing to class
    def __init__(self):
        super().__init__()  # Using super()

# main.py
from player import Player

me = Player()

This way, you can have your own methods inside your class and still use the original library functions.
