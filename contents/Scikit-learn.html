LinearRegression:  doc

We are using a univariate regression. This is a regression with a single explanatory variable.  (Explanatory variables are also referred to as features in machine learning terminology.)

Attention: This library won't draw the regression line for you, It just calculates its properties for you to make you able to predict further predictions. You have to draw it with other libraries like seaborn, etc.

To draw linear regression we need its Linear Equation as "ax + b = y". 

x : explanatory variable a :  slope   b :  y-intercept

from sklearn.linear_model import LinearRegression

regression = LinearRegression()
