Flask_CKEditor

CKEditor integration for Flask, add text editor, add support to image upload, code syntax highlighting and more.  Docs

You write in text-editor field with various styles and colors, then in the output, it gives you the HTML code of your styled text.


from flask_ckeditor import CKEditor
from flask_ckeditor import CKEditorField

ckeditor = CKEditor(flask_app)


class CreatePostForm(FlaskForm):
    title = StringField("Blog Post Title", validators=[DataRequired()])
    body = CKEditorField("Blog Content", validators=[DataRequired()])  ###
    submit = SubmitField("Submit Post")

HTML codes should be similar to these: 

  
   <div>
    {{ ckeditor.load() }}
    {{ ckeditor.config(name='body') }}
    {{ wtf.quick_form(form) }}
  </div>
Then we can save this data to the database the same we behave with the other fields in the form. To be able to show our text again in the way we styled it, we should use a jinja filter called 'safe': 
 <div>
     {{ post.body|safe }}  # if we do not use safe, all html elements would be shown.
</div>
