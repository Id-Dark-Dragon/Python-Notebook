Get hold of values of Function variables(local) and nested functions

You can not hold function variables in the way we behave with class variables(attributes):

class Class1:

    def __init__(self):
        self.name = None
        self.age = None
        self.fun1()

    def fun1(self):
        self.name = 'Ali'
        self.age = 20

print(Class1().name)

result>>>> Ali

def fun1():
    name = 'Ali'
    age = 20

print(fun1().name)

result>>>> AttributeError: 'NoneType' object has no attribute 'name'

instead, you can use these ways:

def fun1():
    name = 'Ali'
    age = 20
    return age, name

age, name = fun1()
print(name)

result>>>> Ali

def fun1():
    name = 'Ali'
    age = 20
    return age, name

a = fun1()  # this gives us a tuple made of (age , name)
print(a[1])

result>>>> Ali

def fun1():
    fun1.name = 'ali'
    fun1.age = 20

fun1()
print(fun1.name)

result>>>> Ali

and here is how to hold values from nested functions:

def fun1():
    def name():
        return 'Ali'
    def age():
        return 20
    fun1.age = age
    fun1.name = name


fun1()
print(fun1.name())

result>>>> Ali

Python Decorators

Decorators are a very powerful and useful tool in Python since it allows programmers to modify the behavior of a function or class. Decorators allow us to wrap another function in order to extend the behavior of the wrapped function.

decorator function itself needs to get a function in the argument's place. wrapper function takes the same arguments as the wrapped function. then we just write the the name of the decorator above the function name with a '@' to work.

This decorator gets the function and its arguments, then gives out the function's name and then initializes it:


def decorator(function):
    def wrapper(*args):
        print(f'{function.__name__}{args}')
        function(*args)
    return wrapper


@decorator
def a_func(*args):
    print(sum(args))


a_func(1, 2)

result >>>>
a_func(1, 2)
3

This is a bit complicated version :


class User:
    def __init__(self, name):
        self.name = name
        self.is_logged_in = False


def is_authenticated_decorator(function):
    def wrapper(*args, **kwargs):
        if args[0].is_logged_in == True:
            function(args[0])
    return wrapper


@ is_authenticated_decorator
def creat_post(user):
    print(f"This is {user.name}'s new post")


user = User('Ali')
user.is_logged_in = True
creat_post(user)

result >>>> This is Ali's new post
