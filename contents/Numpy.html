Numpy

NumPy is a library for the Python programming language, adds support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.  Docs - Beginners guide 

NumPy Arrays: They are like Python lists but they can be used in mathematical calculation more efficiently, they also consume less memory than a normal list, while a Python list can contain different data types within a single list, all of the elements in a NumPy array should be homogeneous. They can be multi-dimensional like a table. Working with them is more and less the same as working with Python lists. Array in visual

Making Arrays:


array_1d = np.array([1.1, 9., 8.1, 4.])    # Creating a 1 dimensional array. (vector)
array_2d = np.array([ [1, 2, 3, 9],
                      [5, 6, 7, 8] ])      # Creating a 2 dimensional array. (matrix)
array_3d = np.array([ [[0, 1, 2, 3],       # Creating a 3 dimensional array or higher. (tensor)
                       [4, 5, 6, 7]],

                        [[7, 86, 6, 98],
                        [5, 1, 0, 4]],

                        [[5, 36, 32, 48],
                        [97, 0, 27, 18]] ])

array = np.arange(10, 30, 3)              # Creating a vector a with values ranging from 10 to 29 with steps of 3.
array = np.random.random((128, 128, 3))   # Generate a 3x3x3 array with random numbers.
array = np.linspace(0, 100, num=9)        # Create a vector of size 9 with values from 0 to 100(included)

Displaying Array Info:


array.ndim                                 # Gives the number of dimension of array.
array.shape                                # Shows rows and columns of the array.
array.size                                 # Find the total number of elements in the array.

Evoke Arrays Data:


array_3d[2, 1, 3]                          # Accessing "18" in array above.

array_2d[0,::2]                            # Gives "[1 3]"

np.nonzero(array)                          # Gives a tuple of non-zero values.

Operations on Arrays:     (more about Matrix multiplication)

a = np.array([ [2, 1, 3, 4],
               [5, 6, 7, 8] ])
b = np.array([1, 2, 3, 9])


a + b
>>>>result    [[ 4  2  6  8]
               [10 12 14 16]]


a * 5          # "5" is called a scaler, and it affects on each element in array.


a * b
>>>>result    [[ 4  1  9 16]
               [25 36 49 64]]


# Matrix multiplication
np.matmul(a, b)
a @ b
>>>>result     [ 49 110]


IMPORTANT FUNCTIONS


"WHERE"  is a function in the NumPy library that returns elements chosen from x or y depending on a given condition.


np.where(condition, output_for_true, output_for_false)

np.where(a < 5, a, 10 * a)


Working With Images: 

Images are just 3D arrays that hold data of pixels RGB in their axis number 2. here we work with NumPy and matplotlib together.


# Converting the image to black and white
"""this formula only works if our red, green and blue values are between 0 and 1 - namely in sRGB format -. 
Currently the values in our img range from 0 to 255 - called RGB format -."""

grey_vals = np.array([0.2126, 0.7152, 0.0722])
srgb = img / 255                   # converting to sRGB
grey_img = srgb @ grey_vals
plt.imshow(grey_img, cmap='gray')  # from matplotlib.pyplot


# Image Rotations
plt.imshow(img[::-1,::-1,:])         # flip
plt.imshow(np.rot90(np.rot90(img)))  # flip

plt.imshow(img.transpose(1,0,2))     # 90 rotation
plt.imshow(np.rot90(img))            # 90 rotation


# Invert (solarize) the colours.
"""To do this you need to convert all the pixels to their "opposite" value, so black (0) becomes white (255)."""
plt.imshow(255-img)
