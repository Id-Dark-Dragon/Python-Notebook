Datetime & Time

*DATETIME

The datetime module in Python supplies classes for manipulating dates and times. It provides 4 main objects for date and time operations: datetime, date, time and timedelta. docs - brief doc

datetime:


# Get the current date and time
now = datetime.datetime.now()
print(now)
print(now.year)
print(now.second)

result >>>>
2023-07-25 19:59:19.161180
2023
19


# Format datetime object as string
print(now.strftime("%m-%d %H"))

result >>>>
07-25 19


# Get current date
print(datetime.date.today())

result >>>>
2023-07-25


# Get date from a timestamp
timestamp = 1627237200
print(datetime.datetime.fromtimestamp(timestamp))

result >>>>
2021-07-25 21:50:00


date:


# Create a date object
date1 = datetime.date(2023, 7, 25)
date2 = datetime.date(2023, 7, 30)

# Get difference between two dates
delta = date2 - date1
print(delta.days)

result >>>>
5


**TIME

The time module in Python provides various time-related functions. It supplies functions for handling time-related tasks such as reading the current time, formatting time, sleeping for a specified number of seconds and so on.  docs


# Sleep for 2 seconds
time.sleep(2)


# Get current time in the format of timestamp
current_time = time.time()
print("Current time:", current_time)

result >>>>
Current time: 1690307751.4404945


# Get current local time based on computer time
local_time = time.localtime(current_time)
print("Local time:", local_time)

result >>>>
Local time: time.struct_time(tm_year=2023, tm_mon=7, tm_mday=25, tm_hour=21, tm_min=25, tm_sec=51, tm_wday=1, tm_yday=206, tm_isdst=0)


# Get current UTC time
utc_time = time.gmtime(current_time)
print("UTC time:", utc_time)

result >>>>
UTC time: time.struct_time(tm_year=2023, tm_mon=7, tm_mday=25, tm_hour=17, tm_min=55, tm_sec=51, tm_wday=1, tm_yday=206, tm_isdst=0)


# Convert a string to a struct_time
time_string = "20 Jun 93"
struct_time = time.strptime(time_string, "%d %b %y")
print("Struct time:", struct_time)

result >>>>
Struct time: time.struct_time(tm_year=1993, tm_mon=6, tm_mday=20, tm_hour=0, tm_min=0, tm_sec=0, tm_wday=6, tm_yday=171, tm_isdst=-1)


# Convert a struct_time to a string
time_string = time.asctime(struct_time)
print("Time string:", time_string)

result >>>>
Time string: Sun Jun 20 00:00:00 1993
