Turtle

Turtle is a Python module that provides a drawing board-like feature, which enables users to create pictures and shapes. Docs

To See Some Basics, Look At The Following Code Which Draws A Random Shape:

from turtle import Turtle, Screen

# SCREEN WORKS
screen = Screen()
screen.setup(width=500, height=400)    # Turtle starting point would be at (250, 200)


# MAKING ONE TURTLE
tim = Turtle(shape="turtle")

tim.color("cyan", "red")
tim.speed("slowest")
tim.pensize(10)     # line width

# 1.    Puts a dot with radius of 20 and goes to wanted location.
tim.dot(20)
tim.goto(0, 30)

# 2.    Makes a dashed-like line.
tim.forward(50)
tim.penup()
tim.forward(50)
tim.pendown()
tim.forward(50)

# 3.    Turns 100 degrees to the left and goes 100 units, then draws a circle.
tim.left(100)
tim.forward(100)
tim.circle(50)

# 4.    Sets the head 20 degrees from the horizon and goes.
tim.setheading(20)
tim.forward(100)

# 5.    Backs to first place and clears the screen
tim.home()
tim.clear()

screen.exitonclick()

    Controlling The Turtle (after the program initialization):
    

def go_up():
    tim.setheading(90)
    tim.forward(10)

screen.listen()
screen.onkey(go_up, "Up") 


    Prompt For Input:
    

name = screen.textinput(title="title", prompt="enter your name")


    Writing On Screen: 
    

tim.write("text", align="center", font=("arial", 80, "bold"))
