Seaborn

Seaborn is a Python data visualization library based on matplotlib. It provides a high-level interface for drawing attractive and informative statistical graphics. 

Here we use the Express module of this lib which can speed up the workflow of chart drawing. Express Examples

import plotly.express as px

PLOTS:

Here are some examples of different plots we can make and some configurations. for more, check Examples in the gallery of the docs.

# Some examples of plots with some of their properties.

bar = px.bar(x=series_a.index,
             y=series_a.values,
             barmode='group',                    # if you had two values in "y" for each "x", this can help.
             title='Genres',
             hover_name=series_a.index,          # the name that is shown when you hover in it.
             orientation='h',                    # For making the bars, horizontal
             color=series_a.values,              # Configures the intensity of color based on the amount of value.
             color_continuous_scale='Agsunset')  # sets a pallet of colors based on what you specify

scatter = px.scatter(merged_df,                  # place DF directly
                     x='App',                    # column name
                     y='Installs',
                     title='Category',
                     size='App',                 # size of each circle in a scatter plot.
                     color='Installs',
                     color_continuous_scale='aggrnyl')

pie = px.pie(labels=series_a.index,
             values=series_a.values,
             title="Content Rating",
             names=series_a.index,
             hole=0.6)                           # makes a hole in the circular plot.(like a donut)

box = px.box(df_apps,
             y='Installs',
             x='Type',
             color='Type',
             notched=True,                       # Makes the plot a bit prettier.
             points='all' )                      # Makes all points visible near the boxplot.

box.show()                                       # should be used to show the graph.

There are two functions that make the plot personalized: "update_layout()" & "update_trace"

"""
a trace is a collection of data points that are plotted together on a graph. 
It is a single set of data that is plotted on a graph, such as a line, scatter, or bar chart. 

A layout is the visual representation of the graph, 
It is used to define the overall look and feel of the graph.
Some of the layout properties include the title, axis labels, font, margin, and color. 
"""

fig.update_traces(textposition='outside',
                  textinfo='percent+label',     # percentage show is usually used for pie plots.
                  textfont_size=15)

fig.update_layout(xaxis_title='Genre',
                  yaxis_title='Apps',
                  coloraxis_showscale=False,   # shows a guid for colors in the scale.
                  yaxis=dict(type='log'),      # makes the scale to raise like "log" scales rather than leaner.
                  xaxis={'categoryorder': 'total descending'}) # sorts the columns of the graph in the order
