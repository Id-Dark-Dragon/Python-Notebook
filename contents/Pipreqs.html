Pipreqs

Pipreqs scans all .py files in your project folder and makes a requierments.txt based on 'import' statements.Â  Docs

Use this commands in cmd based terminal:

pip install pipreqs                #No need to install it on project folder.
(py vir2) C:\DEVELOPMENT\py vir2>pipreqs "E:\Tutorials\Code\Advance\PRJ" 

#ATENTIONS: "" around location.

Notice that if *.py files have an error, pipreqs won't create the requirements file.
