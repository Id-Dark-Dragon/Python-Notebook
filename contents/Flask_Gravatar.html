Flask_Gravatar

This makes it possible to have a random profile picture for each user individually.  Docs 


from flask_gravatar import Gravatar

gravatar = Gravatar(app,
                    size=100,
                    rating='g',
                    default='retro',
                    force_default=False,
                    force_lower=False,
                    use_ssl=False,
                    base_url=None)

It is like a jinja filter in HTML code and every time you give it identifying data of a user (for example: email, hashed password...), it will serve you the same image: 

  
<img src="{{ comment.userstable.name | gravatar(size=200, rating='x') }}" class="commenterImage">
