Working With Dictionaries

MAKING DICTIONARIES:

By "dict" Function:

# Create a dictionary from a list of tuples
data = [('a', 1), ('b', 2), ('c', 3)]
d = dict(data)


# Create a dictionary from a list of lists
data = [['a', 1], ['b', 2], ['c', 3]]
d = dict(data)


# Create a dictionary from keyword arguments
d = dict(a=1, b=2, c=3)                           # Keys won't need to be inside quotation marks.


# THEY ALL RESULT THE SAME:
result >>>> {'a': 1, 'b': 2, 'c': 3}

Using Two Separate Lists:

num = [1, 2, 3, 4, 5]
alp = ['a', 'b', 'c']

# Using for loop
dict1 = {}
for num, alp in zip(num, alp):     # "zip" makes multiple lists iterable at the same time.
    dict1[num] = alp

# Using dictionary comprehension
dict1 = {num: alp for num, alp in zip(num, alp)}

# Using "dict" function
dict1 = dict(zip(num, alp))


print(dict1)

result >>>>  {1: 'a', 2: 'b', 3: 'c'}

ADDING, UPDATING, REMOVING:

dict2 = dict(Sam=1, Ali=2)

# Add or Update an item in the dictionary
dict2["Ali"] = 3

# Delete an item from the dictionary
del dict2['Ali']

ITERATE OVER A DICTIONARY:
for key, value in dict3.items():
