OS

The OS module in Python provides functions for interacting with the operating system.  Docs

OS.ENVIRON:

It is one of its functions that allows us to work with environmental variables.


# Making or Modifying a Var.:    (you can also use .env to make environ vars using dotenv package)
os.environ['JAVA_HOME'] = '/home / ihritik / jdk-10.0.1'

# Catching All in a Dictionary:
pprint.pprint(dict(os.environ), width = 1)

# Accessing One:
java_home = os.environ.get('JAVA_HOME')

# Throwing Something else if it Couldn't Find the Var. (like a string):
os.environ.get('MY_HOME', "Environment variable does not exist")
os.environ.get("DATABASE_URL",  "sqlite:///blog.db")
