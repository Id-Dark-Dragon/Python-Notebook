SQLite3

SQLite is a self-contained, file-based SQL database. SQLite comes bundled with Python and can be used in any of your Python applications where you need to communicate with database. Docs

It is so easy to get in touch with it.

Creating a Database and a Table or Connecting to one:

import sqlite3

# Connect to a SQLite Database or Make a New Database If the database doesn't exist:
conn = sqlite3.connect('database.db')

# Create a Cursor Object to be able to execute SQL commands:
cursor = conn.cursor()

# Create a table using SQL commands:
cursor.execute("""CREATE TABLE IF NOT EXISTS mytable
                   (id INTEGER PRIMARY KEY,
                   name TEXT, 
                   age INTEGER,
                   score REAL)""")   # Use REAL instaed of float

Perform Database Operations:

# Insert Data into the table:
cursor.execute("INSERT INTO mytable (name, age) VALUES (?, ?)", ('Alice', 30))

# Retrieve Data:
cursor.execute("SELECT * FROM mytable")
data = cursor.fetchall()
for row in data:
    print(row)   # Each row prints a tuple of data.

Do not forget to commit data after inserting and close the connection after finishing the job:

conn.commit()

conn.close()
