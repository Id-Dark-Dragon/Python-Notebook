Beautiful Soup

Beautiful Soup is a Python library for pulling data out of HTML and XML files. It works with your favorite parser to provide idiomatic ways of navigating, searching, and modifying the parse tree. docs

Some General Guides:


"""Prettify will add indentation to the soup.""" 
soup.prettify()

___________________________
''' You can access the first tag of any kind by using .p or .a or .h1 etc.'''
soup.title
# <title>The Dormouse's story</title>

soup.title.name
# u'title'

soup.title.string
# u'The Dormouse's story'

soup.title.parent.name
# u'head'


soup.p
# <p class="title"><b>The Dormouse's story</b></p>

soup.p['class']
# u'title'

soup.a
# <a class="sister" href="http://example.com/elsie" id="link1">Elsie</a>

___________________________
"""Getting all the texts from page"""
soup.get_text()

___________________________
''' You can create a list of all occurrences of tag with find_all() '''
soup.find_all('a')
# [<a class="sister" href="http://example.com/elsie" id="link1">Elsie</a>,
#  <a class="sister" href="http://example.com/tillie" id="link3">Tillie</a>]

all_anchor_tags = soup.find_all(name="a")
for tag in all_anchor_tags:
    # getText() gets the text in the tag.
    print(tag.getText())
    # get() can get the value of any tag attribute.
    print(tag.get("href"))

___________________________
''' find() works like find_all() but just gets the first occurrence.'''
soup.find(id="link3")
# <a class="sister" href="http://example.com/tillie" id="link3">Tillie</a>

#You can also find by class_ name.
section_heading = soup.find(name="h3", class_="heading")

___________________________
'''find() can be chained with other find()s.'''
soup.find("head").find("title")
# <title>The Dormouse's story</title>

___________________________
'''You can also find by using a CSS Selector:'''
soup.select_one(selector=".company a")
soup.select("a")
