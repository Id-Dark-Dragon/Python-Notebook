Pyinstaller

Pyinstaller helps you to make the executable file of your python project which can be used in every computer without installing python. Docs - Real python entry

After the first use, it generates a ".spec" file which is a python file that calls Pyinstaller API, It is customizable and makes the future executes faster.

Basic File Making Prompt Inside Shell:


pyinstaller  --name 'myapp' main.py   # "__name" is an option name the app
# OR
pyinstaller myapp.spec

Options:


--onefile                          # Package your entire application into a single executable file.
--hidden-import=selenium           # List multiple top-level imports that PyInstaller was unable to detect automatically.
--add-data folder/version.txt      # Instruct PyInstaller to insert additional data files into your build.
--exclude-module=pytest            # Exclude some modules from being included with your executable.
-w                                 # Avoid automatically opening a console window.
